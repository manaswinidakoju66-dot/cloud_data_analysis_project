# cloud_data_analysis_project
End-to-end data analysis project using Azure, GCP BigQuery, and Looker Studio
# Cloud-Based Data Analysis Project

##  Overview

This project demonstrates an end-to-end cloud data pipeline using:

* Microsoft Azure (Data Storage)
* Google Cloud Platform - BigQuery (Data Processing)
* Looker Studio (Data Visualization)

---

##  Architecture

Data Source → Azure Storage → BigQuery → SQL Analysis → Looker Studio Dashboard

---

##  Azure Implementation

* Created Azure Storage Container
* Uploaded dataset (CSV)
* Managed data in cloud environment

---

##  GCP BigQuery Implementation

* Created dataset and tables
* Imported data into BigQuery
* Performed SQL analysis:

  * Sales by Region
  * Category Performance
  * Customer Insights

---

##  Dashboard (Looker Studio)

* KPIs: Total Sales, Avg Sales, Total Orders
* Regional Sales Analysis
* Category Distribution
* Customer Contribution (%)
* Interactive Filters

---

## Key Insights

* South region contributes highest revenue
* Electronics dominates sales (~90%)
* Top customers contribute major revenue share

---

##  Project Files

* `/azure` → Azure screenshots
* `/gcp` → SQL queries & BigQuery outputs
* `/looker_dashboard` → Dashboard visuals
* `/presentation` → Final PPT

---

##  Outcome

Developed a complete cloud-based data analysis pipeline transforming raw data into actionable insights.
